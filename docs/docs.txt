Сначала мы читаем строку исходного кода программы

(1) Токенизатор:
Токенизатор получает строку, исходный код нашего файла, и возвращает массив 
токенов (структур в которых
содержится инфа о каждом литерале в коде)

(2) Парсинг:
Парсер получает массив токенов и на основании их создает синтаксическое дерево - 
структуру, которая описывает логику нашей программы

(3) Генерация Кода:
Генератор получает корень Синтаксического дерева и, проходясь по нему
генерирует код (тоесть строку в assembly)

Далее мы компилируем код ассемблера с помощью nasm, создавая out.o
И в конце линкуем out.o, создавая файл, доступный для OS
